# صفحه لاگین زیبا - فروشگاه طلا و جواهر

## 🎨 ویژگی‌های طراحی

### ✨ طراحی مدرن و زیبا
- **پس‌زمینه عکس طلا**: استفاده از عکس `gold-is-money-gold-bars-gold-shop-gold.jpg` به عنوان پس‌زمینه
- **افکت شیشه‌ای**: کارت لاگین با افکت blur و شفافیت
- **رنگ‌بندی طلایی**: استفاده از رنگ‌های طلایی (#d4af37) برای تم اصلی
- **انیمیشن‌های نرم**: انیمیشن‌های fade و slide برای تجربه کاربری بهتر

### 📱 Responsive Design
- **موبایل**: طراحی بهینه برای صفحات کوچک
- **تبلت**: نمایش مناسب در اندازه متوسط
- **دسکتاپ**: نمایش کامل با دو ستون

## 🔐 قابلیت‌های صفحه

### ورود (Login)
- **ایمیل**: فیلد ورودی ایمیل با validation
- **رمز عبور**: فیلد رمز عبور با قابلیت نمایش/مخفی کردن
- **دکمه ورود**: با انیمیشن loading
- **پیام‌های خطا**: نمایش خطاهای validation

### ثبت‌نام (Register)
- **نام و نام خانوادگی**: فیلد الزامی
- **ایمیل**: فیلد الزامی با validation
- **شماره تلفن**: فیلد اختیاری
- **نام فروشگاه**: فیلد اختیاری
- **آدرس**: فیلد اختیاری
- **رمز عبور**: فیلد الزامی با validation
- **تکرار رمز عبور**: فیلد الزامی با مقایسه

## 🎯 ویژگی‌های فنی

### Validation
- **فیلدهای الزامی**: بررسی پر بودن فیلدهای ضروری
- **فرمت ایمیل**: validation فرمت صحیح ایمیل
- **طول رمز عبور**: حداقل ۶ کاراکتر
- **تطبیق رمز عبور**: مقایسه رمز عبور و تکرار آن

### انیمیشن‌ها
- **Fade In**: انیمیشن ورود متن‌ها
- **Slide In**: انیمیشن ورود کارت
- **Hover Effects**: انیمیشن‌های hover برای دکمه‌ها
- **Loading**: انیمیشن loading برای عملیات‌ها

### امنیت
- **رمز عبور مخفی**: نمایش نقطه به جای کاراکترها
- **Toggle Password**: قابلیت نمایش/مخفی کردن رمز عبور
- **Session Management**: مدیریت نشست کاربر

## 📱 Responsive Breakpoints

- **xs**: < 600px (موبایل)
- **sm**: 600px - 960px (تبلت)
- **md**: 960px - 1280px (دسکتاپ کوچک)
- **lg**: > 1280px (دسکتاپ بزرگ)
